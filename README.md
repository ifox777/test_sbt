## Описание проекта

Этот проект содержит Go-приложение, в котором была устранена уязвимость **GO-2024-2554**. Сборка проекта выполняется с использованием **мультистейджевого Dockerfile**, а финальный образ минимизирован.

## Устранение уязвимости

Для выявления и устранения уязвимости был использован инструмент **govulncheck**. После внесения исправлений был выполнен повторный анализ, подтверждающий устранение уязвимости.

Отчет прилагается в файле **govulncheck-report.txt**.

## Инструкция по сборке Docker-образа

Проект использует `Makefile` для удобной сборки и запуска.

#### Сборка проекта

```
make build
```

#### Запуск контейнера

```
make run
```

### Использование Docker Compose

Для удобного управления контейнерами используется `docker-compose`.

#### Запуск и сборка контейнера с помощью Docker Compose

```
make up
```

#### Остановка контейнера

```
make down
```

### Подготовка зависимостей для оффлайн-сборки

Для обеспечения оффлайн-сборки используется **vendor.**

Dockerfile использует **мультистейджевую сборку**:

1. **Стадия сборки:** Используется образ `golang `для компиляции Go-приложения.
2. **Стадия финального образа:** Используется `alpine `для минимального размера.
